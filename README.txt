픽셀패스 — 컴퓨터그래픽기능사 필기 합격노트

실행 방법
1. index.html을 더블클릭합니다.
2. 별도 설치나 인터넷 연결은 필요하지 않습니다.

주요 기능
- 4과목 전 범위 핵심노트와 시험 전략
- 15분 집중 루틴과 5단계 합격 로드맵
- 능동회상 암기카드와 학습 난이도 기록
- 10문항 빠른 퀴즈 / 60문항 실전 CBT / 오답 재도전
- 오답 자동 저장, 과목별 숙련도, 진도 자동 저장
- 시험 직전 암기 문장 모음

학습 기록은 현재 브라우저에만 저장됩니다. 브라우저 데이터 삭제 또는
앱 오른쪽 위 초기화 버튼 사용 시 기록이 지워집니다.
